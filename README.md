# Data Analysis
Анализ данных на Python. Программа матпрактикума по математической статистике

Решение задач на визуализацию основных результатов математической статистики и случайных процессов. 

### Темы математической статистики: 
- ОМП
- доверительные интервалы 
- регрессия
- критерии проверки простой гипотезы
- проверка принадлежности параметрическому семейству
- проверка однородности
- проверка независимости


### Главы случайных процессов:
 - Марковские цепи и процессы
 - ветвящиеся процессы
 - Пуассоновские потоки
